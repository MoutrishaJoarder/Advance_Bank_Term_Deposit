# Advance_Bank_Term_Deposit
Conducted exploratory data analysis on bank marketing data to examine customer demographics, balance trends, and campaign effectiveness; used Python (Pandas, Matplotlib, Seaborn) to uncover patterns influencing term deposit subscriptions.
This project explores a bank marketing dataset to understand customer characteristics and identify factors influencing term deposit subscriptions.
The analysis includes exploratory data analysis (EDA), visualization, and basic predictive modeling using Python.

## Objectives

- Analyze customer demographics
- Examine campaign effectiveness
- Study call duration and subscription behavior
- Explore correlations among numerical variables
- Build a Logistic Regression model for prediction

## Dataset

Source:
UCI Bank Marketing Dataset

The dataset contains customer information including:

- Age
- Job
- Marital Status
- Education
- Housing Loan
- Personal Loan
- Contact Type
- Campaign Details
- Call Duration
- Previous Campaign Information
- Subscription Status

## Key Insights

- Most customers did not subscribe to a term deposit.
- Customers with longer call durations were more likely to subscribe.
- Most campaigns contacted customers only a few times.
- Customer demographics and previous campaign information influenced subscription behavior.
